功能介绍：
火狐浏览器扩展，自带本地播放器版，去除国内主流视频网站的几十秒无节操广告，
如爱奇艺，pps，letv，pptv，优酷，酷六，腾讯，搜狐，17173，多玩等

使用方法：
下载的扩展拉到火狐附加组件列表即可安装，关闭打开的视频页并清理浏览器缓存，
再重启下浏览器，看看是不是广告影无踪了
